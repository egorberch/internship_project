# Проект по практике на C#

Консольное приложение на C# (.NET 10) для работы с базой данных PostgreSQL через Dapper. 
Таблицы и тестовые данные создаются автоматически при старте.

## Задание

Есть три таблицы: Пользователи, Роли, и связка Пользователь-Роль (многие ко многим). У каждого пользователя есть хотя бы одна роль.

Необходимо решить две задачи:
1. Вывести всех пользователей и роли, в которых они состоят (используя LEFT JOIN).
2. По каждой роли определить количество пользователей, которые входят в эту роль (используя INNER JOIN).

## Инструкция по запуску

### 1. Подготовка базы данных
Убедитесь, что у вас установлен и запущен PostgreSQL, и создана пустая база данных (например, `internship_db`). 
Вручную выполнять SQL-скрипты **не нужно** — программа сделает это автоматически при первом запуске (с помощью KingMigrations).

### 2. Настройка подключения
Создайте в корне проекта файл `.env` (вы можете скопировать его из `.env_example`) и укажите ваши актуальные параметры:
```env
DB_HOST=localhost
DB_PORT=5432
DB_NAME=internship_db
DB_USER=postgres
DB_PASSWORD={ваш_пароль}
```

### 3. Запуск приложения
Откройте терминал в корневой папке проекта, перейдите в папку с приложением и запустите проект:
```bash
cd InternshipApp
dotnet run
```
Далее следуйте инструкциям в интерактивном меню программы.
