# Проект на практику на C#

Консольное приложение на C# (.NET 10) для работы с базой данных PostgreSQL через Dapper. 
Таблицы и тестовые данные создаются автоматически при старте.

## Структура проекта
```text
internship_project/
├── InternshipApp/
│   ├── Migrations/
│   ├── Models/
│   └── Program.cs
├── .env_example
├── .gitignore
└── README.md
```

## Инструкция по запуску

### 1. Подготовка базы данных
Убедитесь, что у вас установлен и запущен PostgreSQL, и создана пустая база данных (например, `internship_db`). 
Вручную выполнять скрипты **не нужно** - программа сделает это автоматически при старте (KingMigrations).

### 2. Настройка подключения
Создайте в корне проекта файл `.env` (можно просто скопировать из `.env_example`) и укажите ваши параметры:
```env
DB_HOST=localhost
DB_PORT=5432
DB_NAME=internship_db
DB_USER=postgres
DB_PASSWORD={пароль}
```

### 3. Запуск
Откройте терминал в папке с приложением и запустите проект:
```bash
cd InternshipApp
dotnet run
```
Следуйте инструкциям в интерактивном меню.
